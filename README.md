# Projet Validation et Contrôle Qualité des Données

Conception et mise en œuvre d’un pipeline de scoring de crédit pour l’évaluation du risque de défaut, incluant des mécanismes robustes de validation et d’assurance qualité des données.

## Contenu
- `projet_validation_UA2.ipynb` : notebook principal
- `tests_config.json` : fichier de configuration des tests
- analyses de qualité, modélisation, fairness et détection de drift

## Étapes du projet
1. Contrôle qualité des données
2. Validation de l’ingénierie des caractéristiques
3. Validation du modèle
4. Monitoring et détection du drift

## Outils utilisés
- Python
- Pandas
- Scikit-learn
- Seaborn / Matplotlib
- SciPy

## Résumé
Le projet met en place un pipeline de qualité des données appliqué à un dataset de risque de crédit, avec nettoyage, tests unitaires, modélisation, validation croisée, analyse d’équité et simulation de data drift.
